# REST API
REST(웹을 위한 아키텍쳐 스타일) 아키텍쳐(제약조건의 집) 스타일을 따르는 API 
-> REST스타일의 제약조건 스타일을 따르는것 
## REST 스타일의 구성요소
- client - server
- stateless
- cache
- uniform interface
    uri로 식별이 되야한다.
- layered system
- code-on-demand(optional)
